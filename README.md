# 國小五年級下學期數學（6-10單元）學習障礙學生專用總複習簡報

本專案是一個專為學習障礙（如讀寫困難、計算障礙、注意力缺陷等）學生設計的**網頁版互動複習簡報**。透過多重感官支持（語音朗讀）、顏色編碼公式、步驟拆解、3D 幾何展開與拖動百分率等互動，將抽象的數學概念轉化為具體的視覺與觸覺體驗。

---

## 🌟 核心特色功能

1. **多重感官語音朗讀 (TTS)**：
   - 整合瀏覽器內建語音合成（Web Speech API），提供清晰、親切的繁體中文語音引導。
   - 語音播放速度適度放慢（0.85倍速），方便理解能力較弱或讀寫障礙的學生吸收。
   - 每張投影片設有「獨立喇叭按鈕」，可隨時單獨朗讀題目或解析。

2. **數學公式與結構顏色編碼**：
   - **藍色**：整數 / 時間「時」/ 分母 / 全體數量
   - **紅色**：小數 / 時間「分」 / 分子 / 部分數量
   - **粉色**：計算單位 / 立體圖形側邊
   - **綠色**：最終答案 / 成果提示

3. **立體圖形 3D 展開互動**：
   - 內建 3D 渲染長方體。點擊按鈕，長方體會平滑地「在空間中展開」成 2D 網格平面，對應的面使用相同顏色標記，幫助學生直觀理解表面積公式：`(長×寬 + 長×高 + 寬×高) × 2`。

4. **百分率百格圖與位值滑動**：
   - **百格圖互動**：提供 10x10 格網格，拖動滑桿可即時填色並動態計算小數、分數與百分率（%），將比例具體化。
   - **小數點移動動畫**：點擊轉換按鈕，小數點字元會平滑向左或向右跳躍兩格（如 `0.78` ⇄ `78%`），動態演練位值移動法則。
   - **互換公式標示**：明確列出「小數 ➔ 百分率：×100」、「百分率 ➔ 小數：÷100」，語音同步朗讀換算口訣。

5. **時間乘除進退位與化聚圖表雙頁籤**：
   - 針對時間乘除（60進位制）的難點，提供步驟式進位提示（如 `75分 ➔ 1時 15分`），降低計算負荷。
   - 提供「時間計算練習」與「時間化聚圖表」雙頁籤，一鍵切換至由課本取材的「時間變身圖」圖表，支援局部放大與語音朗讀。

6. **線對稱鏡像繪圖網格**：
   - 提供 10x10 的互動對稱網格。學生在左側點選藍色格子，右側會自動鏡像產生綠色對稱格，透過遊戲化操作理解對稱點與對稱軸。

7. **單位化聚資訊圖表檢視器**：
   - 收錄從課本 NotebookLM 來源下載的四份核心化聚圖表（面積單位、長度單位、重量單位、化聚總表）。
   - 支援分頁切換、一鍵放大 1.5 倍與多感官語音朗讀口訣。

8. **溫馨隨堂練習與特教解鎖防跳過機制**：
   - 每個單元複習後皆有 5 題隨堂小測驗，共計 25 題。
   - 測驗頁面中，系統會隱藏「下一頁」並禁用右方向鍵，確保學生完整完成作答才可解鎖下個單元。
   - 答對時釋放全螢幕彩帶煙火，並獲贈黃金星星；答錯時給予溫和的鼓勵語音與鷹架式算理提示。

## 🚀 如何開始使用？

由於本專案採用純前端無伺服器架構，**不需安裝任何額外軟體**：

1. 開啟您的瀏覽器（建議使用 **Google Chrome** 或 **Microsoft Edge** 以獲取最佳的語音合成效果）。
2. 在瀏覽器中按下 `Ctrl + O`，選擇並開啟 `C:\Users\User\.gemini\antigravity\scratch\math-special-education-review\index.html`。
3. 或者直接按兩下雙擊 [index.html](file:///C:/Users/User/.gemini/antigravity/scratch/math-special-education-review/index.html) 即可開始播放！

- 鍵盤 **`←` (左方向鍵)** 或 **`PageUp`**：切換至上一張投影片。
